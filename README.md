# Homebrew Tap for Fastskill

Install [fastskill](https://github.com/gofastskill/fastskill) via Homebrew.

## Installation

```bash
brew install gofastskill/cli/fastskill
```

## Usage

```bash
fastskill --version
```

## Upgrading

```bash
brew upgrade fastskill
```

## Uninstalling

```bash
brew uninstall fastskill
brew untap gofastskill/cli
```
